# our-app — SmartMedia

**SmartMedia** is a complete cross-platform mobile application and system keyboard extension (Android IME + iOS Keyboard) that:

1. Searches open-source GIFs  
2. Checks whether the host app’s text field accepts GIFs / MP4  
3. Transcodes GIFs to universally compatible **H.264 MP4** on-the-fly when GIFs are blocked  
4. Falls back to the native system share sheet when neither format is accepted  

## Quick start

```bash
cd smartmedia
flutter pub get
flutter run
```
